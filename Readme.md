PCB-CNC Software Banlance v1.2
============================================
这是一个借用数学计算的方法将PCB平面虚拟为绝对平面的软件。
主要用於PCB线路雕刻方面。
具体用法如下：

1. 准备好雕刻机（比如AMCNC-01），PCB，夹具或双面胶。
2. 找一根废弃的0.3的钻头，把刀刃部分全部去掉，只剩下刀柄和一小切跟刀刃连接的尖端，稍打磨一下。
3. 将PCB对齐标线，标线的方式可以是激光十字或是自画的十字。
4. 用夹具（强烈建议）固定好，或是用双面胶贴住。
5. 启动Mach3，移动主轴到PCB左下角（如果你想以右下角为基准的话，就移动右下角）。
6. 下刀（不用开主轴）至尖端刚好碰到PCB平面，然后Z轴清零。
7. 本软件的Deviation部分的左下角的数据，同样也填0.
8. 然后，按照本软件左边的图示，将主轴移到相应的位置。
9. 下刀至尖端刚好碰到PCB平面，然后将Z轴位置数据填写到本软件相应的数据位置。
10. 九个点全部测量完毕之后，点击G-Code File，选择你所要平衡的NC文件，然后点Start即可。
11. 当处理结束后，会在该NC文件相同位置产生一个以.pcsb.nc结尾的文件，这个文件就是平衡后的文件。
12. Mach3中，载入此文件（*.pcsb.nc），检查刀路是否有误。
13. 换成雕刻V刀，并移动主轴到PCB左下角，重新对刀（记得现在要开主轴了，不然刀毁），三轴全清零。
14. 升起主轴，准备开工雕刻吧。
15. 警告：在你没有熟悉本软件之前，最好先用废板子来尝试。
